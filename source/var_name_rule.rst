变量标识符的命名规则
================================

.. contents:: 目录


标识符命名规则
--------------------

#. 标识符的第一个字符必须是字母（a-Z）或者下划线_
#. 标识符的其他部分可以是字母（a-Z）或者下划线_或者数字（0-9）组成
#. 标识符大小写敏感

示例::

    >>> /a='1'                                    # 无效
      File "<stdin>", line 1
        /a='1'
        ^
    SyntaxError: invalid syntax
    >>> ?a='1'                                    # 无效
      File "<stdin>", line 1
        ?a='1'
        ^
    SyntaxError: invalid syntax
    >>> _a='1'                                    # 有效
    >>> _a
    '1'

    >>> a1='a1'                                   # 有效
    >>> a1
    'a1'
    >>> a_1="a_1"                                 # 有效
    >>> a_1
    'a_1'
    >>> a_a_1="a_a_1"                             # 有效
    >>> a_a_1
    'a_a_1'
